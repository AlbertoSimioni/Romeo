4. POSSIBLE ERRORS

In this section user can see program's errors along with their meaning.

4.1. ERR 0x2BFD4F16

  • Typology: Error
  • Message: Can't find the following files
  • Meaning: message indicating that the program cant find one or more filesystem's files through the given paths

4.2. ERR 0x117540C0

  • Typology: Error
  • Message: Check at least one Subject and select one Protocol
  • Meaning: message indicating that the dataset has no subject to be analyzed or that no dataset's protocol is selected

4.3. ERR 0x1177F11E

  • Typology: Error
  • Message: Select at least one algorithm or one feature
  • Meaning: message indicating that a protocol must contain at least one feature or an algorithm

4.4. ERR 0x11E2B40C

  • Typology: Error
  • Message: Can't load the dynamic library!
  • Meaning: message indicating that the new feature or the new algorithm defined by user contains an error. If it's the first case (new feature's insertion), error concerns the function name or the type or the file's path. If it's the second case (new algorithm's insertion), error concerns the function name or the file's path or parameters.
