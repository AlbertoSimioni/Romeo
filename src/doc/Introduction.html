1.1. Welcome to ROMEO

The software ROMEO, the Medical Imaging Cluster Analysis Tool, allows users to perform the features extraction and the cluster analysis of biomedical images. The program can show results of the analysis execution in real-time and can save them in a folder chosen by the user. The main characteristics of this product are simplicity and flexibility, in fact users must be able to insert through the GUI new features and algorithms defined by them.

This program is built as a stand-alone application that works in complete independence from other programs. In order to work with ROMEO, user must have a computer running one of the following operating systems: Windows 7 (Service Pack 1), Ubuntu 12.04 LTS, Mac OS X 10.7 Lion (both 32-bit and 64-bit versions).

1.2. Main characteristics

The main characteristics that distinguish ROMEO are:
- Possibility to process two-dimensional and three-dimensional biomedical images and videos.
- Support for many different image/video formats, including Analyze and NIfTI.
- Intuitive and easy to use graphical user interface.
- Possibility to extend the software by adding new features extractor and new clustering algorithms.

1.3. Realization

ROMEO has been realized in C++, a programming language famous for its intrinsic speed due to absence of any virtual machine or garbage collector. It also uses ITK and VTK libraries: these ones provide image manipulation with very efficient constructs. For the graphic it has been used Qt, that is the most famous C++ framework for the definition of user interface. The software has been largely tested through advanced software engineering tools, like CCCC, CppCheck and QTest: this grants robustness to the final product and the reliability of its results.
