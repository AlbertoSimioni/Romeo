7. CONTACT US

In the eventuality that during the interaction with the software ROMEO a problem or a malfunction occurs, the user can notify the group Hurry Up! through the email address: hurryupswe@gmail.com

To help the group understanding the problem, user has to specify the following information:

  • - A relevant subject (such as Error report and its associated code)
  • - A clear and detailed description of the problem and the circumstance when it appeared
  • - The operating system which had the error
  • - Any other information that can be useful to identify the problem (for example a screenshot)
